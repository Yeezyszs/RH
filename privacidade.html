1. Introdução

Esta Política de Privacidade descreve como a [NOME DA EMPRESA] coleta, processa, armazena e protege os dados pessoais de seus colaboradores através do Sistema RH.

O respeito à privacidade e proteção de dados é fundamental para nosso negócio. Operamos em conformidade com a Lei Geral de Proteção de Dados (LGPD) e outras legislações aplicáveis.

2. Seus Direitos (LGPD)

Você tem os seguintes direitos garantidos por lei:

Direito O que é Como exercer
Acesso Solicitar cópia de seus dados Email: dpo@empresa.com
Retificação Corrigir dados incorretos Sistema RH ou solicitação por email
Exclusão Solicitar exclusão de dados* Email: dpo@empresa.com

Oposição Opor-se a certos processamentos Email: dpo@empresa.com

*Sujeito a limitações legais (impostos, saúde ocupacional, etc.)

3. Dados Coletados

O Sistema RH processa os seguintes dados:

✓ Dados Identificadores

  • Nome completo, CPF, email, telefone, RG/CNH

✓ Dados Profissionais

  • Cargo, departamento, data de admissão, matrícula, salário

⚕️ Dados de Saúde (Sensíveis)

  • ASOs, exames médicos, EPIs, treinamentos de segurança

📝 Dados Comportamentais

  • Advertências, avaliações, pesquisas de clima, feedback

💰 Dados Financeiros

  • Salário, benefícios, férias, histórico de pagamentos

4. Como Seus Dados São Protegidos

Medida de Segurança Descrição
🔐 Criptografia Dados criptografados em trânsito (HTTPS) e repouso (AES-256)
🔑 Autenticação Login seguro com senha + MFA para admin/RH
👤 Controle de Acesso Row Level Security: você vê apenas seus dados (por perfil)
📋 Auditoria Logs registram quem acessou quais dados
💾 Backup Automático diariamente com retenção 7 dias
🏢 Operadora Supabase (SOC 2 Type II, GDPR, LGPD compliant)

5. Retenção de Dados

Cada tipo de dado é retido pelo período legalmente necessário:

Tipo de Dado Retenção Por quê?
Dados profissionais Enquanto contratado Necessário para operação
Salários/folha 5 anos pós-desligamento Retenção fiscal/trabalhista
ASOs/exames 40 anos Exigência legal (NR-7)
Treinamentos 5 anos Exigência legal (NRs)
Advertências Até 2 anos Prescrição trabalhista
Feedback/avaliações 1 ano Referência profissional
⚠️ Importante: Após o período de retenção, seus dados são excluídos permanentemente ou anonimizados.

6. Compartilhamento de Dados

Com Quem Compartilhamos

  • ✓ Supabase: Armazenamento de dados (Operadora com DPA assinado)
  • ✓ Fisco: Impostos e DARF (obrigação legal)
  • ✓ INSS: Contribuições e GPS (obrigação legal)
  • ✓ Médicos/Clínicas: Exames de saúde ocupacional (contrato)

Com Quem NÃO Compartilhamos

  • ❌ Empresas de marketing ou publicidade
  • ❌ Redes sociais
  • ❌ Plataformas não conformes LGPD
  • ❌ Terceiros sem base legal

7. Suas Responsabilidades

Para manter a confidencialidade, você se compromete a:

  • Manter sua senha segura (não compartilhe com colegas)
  • Fazer logout ao sair do computador
  • Não acessar dados além do seu escopo de perfil
  • Não divulgar dados de colegas (salário, saúde, avaliação)
  • Relatar incidentes de segurança ao RH
⚠️ Consequências de Violação: Violação de confidencialidade pode resultar em advertência, bloqueio de acesso, dispensa por justa causa ou ações legais.
